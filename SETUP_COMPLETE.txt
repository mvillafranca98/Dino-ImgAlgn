🎯 TUNING FOR BETTER ACCURACY:

Not getting good results? Try these:

1. Lower box threshold for more detections:
   python run_model.py -i image.jpg -t "person" --box-threshold 0.25

2. Raise box threshold for fewer false positives:
   python run_model.py -i image.jpg -t "person" --box-threshold 0.45

3. Adjust text threshold for better label matching:
   python run_model.py -i image.jpg -t "cat" --text-threshold 0.15

4. Use specific, clear prompts:
   ✅ Good: "person . red car . black dog ."
   ❌ Bad: "stuff in image"

See TUNING_GUIDE.md for detailed instructions!

⚡ QUICK REFERENCE:

Parameter          | Default | Description
-------------------|---------|----------------------------------
--box-threshold    | 0.35    | Detection confidence (0.0-1.0)
--text-threshold   | 0.25    | Text matching strength (0.0-1.0)
--save-json        | False   | Save results as JSON
--gpu              | False   | Use GPU (CPU by default)

Lower box-threshold  → More detections (may include false positives)
Higher box-threshold → Fewer detections (only high confidence)
Lower text-threshold → More flexible text matching
Higher text-threshold → Stricter text matching
